Installing Arazzo VSCode takes less than a minute and requires no additional configuration. The extension is published on the [Visual Studio Marketplace](https://marketplace.visualstudio.com/items?itemName=connethics.arazzo-vscode) and installs like any other VS Code extension. Choose whichever method is most convenient for you.

## Requirements

Arazzo VSCode requires **VS Code 1.90.0 or newer**. Before you install, confirm your version under **Help → About** in VS Code and update if needed.

## Method 1: Install from inside VS Code

<Steps>
  <Step title="Open the Extensions panel">
    Press `Ctrl+Shift+X` (Windows / Linux) or `Cmd+Shift+X` (macOS) to open the **Extensions** side panel, or click the Extensions icon in the Activity Bar on the left.
  </Step>
  <Step title="Search for Arazzo VSCode">
    Type **Arazzo VSCode** in the search box at the top of the Extensions panel. The extension published by **connethics** will appear in the results.
  </Step>
  <Step title="Click Install">
    Select the **Arazzo VSCode** result and click the **Install** button. VS Code downloads and activates the extension automatically — no restart required.
  </Step>
</Steps>

## Method 2: Install from the Marketplace website

<Steps>
  <Step title="Open the Marketplace page">
    Navigate to the extension's Marketplace page in your browser:

    ```text
    https://marketplace.visualstudio.com/items?itemName=connethics.arazzo-vscode
    ```
  </Step>
  <Step title="Click Install">
    Click the **Install** button on the Marketplace page. Your browser will prompt you to open VS Code. Confirm the prompt to launch VS Code and begin the installation automatically.
  </Step>
  <Step title="Confirm installation in VS Code">
    VS Code opens the extension detail page and starts installing. Wait for the **Install** button to change to **Uninstall**, which confirms the extension is ready.
  </Step>
</Steps>

## After installation

The extension activates automatically whenever you open any YAML file — no configuration or settings changes are needed. Open an Arazzo specification and the preview icon (eye) and flowchart icon (graph) appear in the editor toolbar immediately.

<Tip>
  Arazzo VSCode is completely free and collects no telemetry, analytics, or usage data. Your API specifications never leave your machine.
</Tip>
