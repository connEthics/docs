Arazzo VSCode is a free VS Code extension by connEthics that gives you everything you need to author, visualize, and validate [Arazzo](https://spec.openapis.org/arazzo/latest.html) API workflow specifications without leaving your editor. Open any Arazzo YAML file and get an interactive documentation preview, a Mermaid-powered flowchart diagram, real-time validation diagnostics, and smart autocompletion — all updating live as you type.

## Get up and running in three steps

Whether you are writing your first Arazzo specification or migrating an existing API workflow, you can go from a blank file to a live visual diagram in minutes.

<Steps>
  <Step title="Install the extension">
    Search for **Arazzo VSCode** in the VS Code Extensions panel and click **Install**. The extension is free and activates automatically whenever you open a YAML file.
  </Step>
  <Step title="Open an Arazzo YAML file">
    Open any existing Arazzo specification or create a new `.yaml` file and start writing one from scratch. The extension detects the Arazzo structure and activates autocompletion and validation immediately.
  </Step>
  <Step title="Launch the Preview and Flowchart">
    Click the eye icon in the editor toolbar to open the **Arazzo Preview**, or click the graph icon to open the **Flowchart** view. Both panels update in real time as you edit your YAML. You can also trigger either view from the Command Palette with `Open Arazzo Preview` or `Open Arazzo Flowchart`.
  </Step>
</Steps>
